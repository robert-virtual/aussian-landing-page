EBOOKS

Este sitio se diseño para proporcionar a los usuarios una experiencia de exploración y descubrimiento de libros en formato digital. Este proyecto demuestra las habilidades en diseño web, desarrollo front-end y back-end, así como nuestra capacidad para crear experiencias de usuario atractivas y funcionales.

Inventario

Gestión de inventario de los productos cuando los usuarios tienen privilegios.

Pasarela

Se implemento una pasarela de pagos segura para procesar las compras. Además la página incluye un carrito donde los usuarios pueden agregar o cancelar los libros a medida que los encuentran.

Método de pago

Una vez que se compra un libro, este se agrega automáticamente a la sección de libros comprados del usuario, permitiendo un fácil acceso y gestión de sus adquisiciones.

Manejo de roles de usuario

El manejo de roles de usuario en un sistema de software permite asignar permisos específicos a distintos tipos de usuarios según sus roles dentro del sistema.
